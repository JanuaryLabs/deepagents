  Popular Text-to-SQL Benchmarks to Consider

  | Dataset    | Size | Difficulty | Description                                             |
  |------------|------|------------|---------------------------------------------------------|
  | Spider     | 10k+ | Hard       | Multi-table, cross-domain. Industry standard benchmark  |
  | WikiSQL    | 80k+ | Easy       | Single-table queries, good for baseline                 |
  | BIRD       | 12k+ | Hard       | Real-world databases with dirty data & domain knowledge |
  | KaggleDBQA | 400  | Medium     | Real Kaggle databases, practical scenarios              |
  | SParC      | 4k+  | Hard       | Multi-turn conversational SQL                           |
  | CoSQL      | 3k+  | Hard       | Dialogue-based SQL with clarifications                  |
  | SEDE       | 12k  | Medium     | Stack Exchange data queries                             |
